# Affordable-Housing-Project
Affordable-Housing-Project This project is about Finding Safe, Verified and Affordable Homes Easily.it aims to solve Fake listings,Unclear rent prices,Time wasted on bad houses,No easy comparison;by enabling Search by budget,Verified listings ,Compare houses,Reviews from tenants

🎯 Purpose:  
To build a responsive affordable housing platform that allows renters to filter listings and landlords to submit housing data using HTML, CSS, JavaScript, and testing tools.

## 🚀 Project Overview

This project is designed to:
- Help renters filter housing based on affordability
- Allow landlords to submit rental listings
- Show last updated/verified dates for transparency

---

## 🎯 MVP Features

### 👤 For Renters
- Filter houses by maximum rent
- Filter by location
- Filter by number of bedrooms

### 🏡 For Landlords
- Submit new rental listings via a form
- Add address, rent, bedrooms, and restrictions

### 📅 Transparency Feature
- Each listing shows last updated date
